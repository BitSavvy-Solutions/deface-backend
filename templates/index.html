📡 API Documentation

Send a multipart POST request to /process with your video and any of the options below. Add the header Accept: application/octet-stream to get the file returned directly instead of an HTML page.

curl -X POST http://localhost:5000/process \
  -H "Accept: application/octet-stream" \
  -F "file=@your_video.mp4" \
  -F "replacewith=mosaic" \
  -F "mosaicsize=20" \
  -F "thresh=0.2" \
  -F "mask_scale=1.3" \
  -F "keep_audio=on" \
  -F "boxes=on" \
  -F "draw_scores=on" \
  -F "keep_metadata=on" \
  -F "scale=640x360" \
  -F "backend=auto" \
  --output anonymized.mp4

Available values:
  replacewith   blur | mosaic | solid | none   (default: blur)
  mosaicsize    5 to 100                        (default: 20, mosaic only)
  thresh        0.01 to 1.0                     (default: 0.2)
  mask_scale    float                           (default: 1.3)
  backend       auto | onnxrt | opencv          (default: auto)
  scale         WxH format e.g. 640x360        (optional)
  keep_audio    on                              (omit to disable)
  boxes         on                              (omit to disable)
  draw_scores   on                              (omit to disable)
  keep_metadata on                              (omit to disable)
